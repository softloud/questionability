# questionability data

This directory contains publication-level datasets ready for analysis. 

- naming reflects the ids that identify a unique row in the data
- request an attribute (i.e., another column, e.g., number of teams who used each column, human-readable label) by opening an issue and labelling it with `wrangling`
- these datasets are ready to be called by scripts in R
- bugs/missingness data issues: label as `provisioning` and `bug`
